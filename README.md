# RACCER
Code for paper: RACCER: Towards Reachable and Certain Counterfactual Explanations for Reinforcement Learning

### Requirements

```{bash}
python = 3.7
pymoo = 0.6.0
tensorflow==1.14.0
torch==1.12.1
```

### Installation

```{bash}
conda create -n raccer
conda activate raccer
pip install -r requirements.txt
```

### Running RACCER

```{bash}
python main.py
```
